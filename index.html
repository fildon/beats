Beats

This project is in active development. Not all features are ready yet.

Instructions:

Each line represents one drum loop. Drum loops should start with a drum type followed by `|` and then the rhythm it should play. If a line does not match this format, it is ignored. The rhythm part of the line can be comprised of any symbols, but only `x` and `o` are treated as hits. Each character represents one 16th of a 4/4 measure (one semiquaver).

Available Drums:
  • HH (Hihat)
  • S (Snare)
  • B (Bass Drum)
  • T1 (Tom 1)
  • T2 (Tom 2)
  • T3 (Tom 3)
Current BPM: 80
