This project is in active development. Not all features are ready yet. At present the drum tab is parsed only at the moment the start button is pressed. At that time, only lines starting with "HH", " S", " B" are considered. For each line, any "x" or "o" is treated as a hit. Each character represents one 16th of a measure (one semiquaver).

Current BPM: 80
